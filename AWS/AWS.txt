


CREATE AN EC2-INSTANCE:

1. CHOOSE LAUNCH INSTANCE 
2. CHOOSE THE INST TYPE ACCORDING TO YOUR APPLICATION NEED. WE HAVE CHOOSEN t2.micro(free)
3. CHOOSE No OF INSTANCE:1
4. CHOOSE THE VOL (DEFAULT 8 GB ROOT VOL)
5. ASSIGN NAME TO YOUR INSTANCE USING TAG
Name: LT-UBUNTU
6.CONFIGURE SEC GROUP

INBOUND: SSH ANYWHERE

7. REVIEW

8. CHOOSE THE KEY-PAIR(EITHER EXISTING OR CREATE A NEW KEY-PAIR)
9. CONNECT USING PUTTY  

DOWNLOAD PUTTY FROM www.putty.org


10. IT WILL ASK FOR AUTHENTICATION.YOU NEED TO GIVE THE .PPK KEY


AFTER THAT YOU WILL BE GET CONNECTED TO THE UBUNTU SERVER:
